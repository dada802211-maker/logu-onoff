# 🔐 React + PHP Session Auth Template

React（フロントエンド）とPHP（バックエンド）で構成した
シンプルなセッション認証付きログインテンプレートです。

---

## 🚀 機能

- ログイン / ログアウト
- PHPセッションによる認証管理
- ログイン状態に応じたメニュー切り替え
- CORS対応（React ⇄ PHP）
- 認証状態チェックAPI（`me.php`）

---

## 🧩 技術スタック

### Frontend
- React
- TypeScript
- react-router-dom

### Backend
- PHP 8.x
- セッション認証
